# 样例简谱（samples）

本目录存放 **可公开** 的中文敬拜风格简谱样例图，用于开发、演示与 PoC 验收。

## 使用约定

| 规则 | 说明 |
|------|------|
| 版权 | 仅提交自制、公有领域或已获授权素材 |
| 格式 | 优先 `png` / `jpg`；单页、印刷清晰者优先 |
| 私有数据 | 放在本地 `samples/private/`（已 gitignore），勿推送 |

## 当前内容

| 文件 | 说明 | 来源 |
|------|------|------|
| `001_poc_digits.png` | PoC 合成样例：数字简谱风格行 + 英文说明文字，供 OpenCV/OCR 冒烟 | 仓库自制（OpenCV 绘制） |

## 建议命名

```text
samples/
  001_printed_clear.png      # 清晰印刷
  002_phone_photo.jpg        # 手机拍照（后续阶段）
```

添加样例时请在本 README 表格中补充来源与授权说明。
